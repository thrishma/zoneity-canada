The Problem

Canada's housing crisis is partly a zoning crisis — but zoning bylaws are buried in hundreds of dense PDFs across dozens of municipalities, with no standard format, no searchability, and no easy way to compare rules across cities. The people who most need this information — housing advocates, developers, planners, journalists — have no good tools to access it.

A developer assessing a site in Kitchener vs. Waterloo needs to read two different 400-page PDFs to find out whether a multiplex is permitted, what the height limit is, or how much parking is required. A housing researcher comparing secondary suite rules across Ontario must manually scan bylaw after bylaw.

There is no national equivalent of a zoning lookup tool for Canada. Zoneity is that tool.

The Solution

Semantic Bylaw Search

Ask plain-English questions across indexed bylaw text. Results ranked by vector similarity — finds the right clause even when exact wording differs.

Structured Metric Extraction

Key zoning values extracted from PDFs automatically using regex + GPT-4o: lot size, height, parking, density, secondary suites, multiplexes.

Municipality Comparison

Side-by-side table of zoning metrics across up to 4 municipalities. Instantly see where regulations are more or less permissive.

AI Chat Agent

LangChain agent with 3 tools: structured metrics, bylaw text search, and open data queries. Answers cite specific bylaw sections.

Waterloo Open Data Integration

167k+ features from data.waterloo.ca: building permits, planning communities, neighbourhood associations, landmarks, address proximity.

Community Submissions

Users submit bylaw PDFs not yet indexed. Admin review queue triggers ingestion pipeline — crowdsourced national coverage expansion.

Metric Extraction Pipeline

Zoneity extracts 6 structured zoning metrics from every ingested bylaw PDF using a two-stage pipeline designed to maximize coverage while minimizing LLM cost:

Metric Description Extraction Method
Minimum Lot Size Smallest residential lot in base R-1 zone (sqm) Regex → LLM fallback
Max Building Height Maximum height in residential zones (metres) Regex → LLM fallback
Min Parking per Unit Required parking spaces per dwelling unit Regex → LLM fallback
Secondary Suites Permitted Whether in-law suites are permitted as-of-right Semantic search + GPT-4o
Multiplex (4+ units) Permitted Whether 4+ unit buildings are permitted in residential zones Semantic search + GPT-4o
Max Density Maximum residential density in medium-density zones (units/ha) Regex → LLM fallback

Who It Serves

Housing Advocates
Quickly identify which municipalities have the most restrictive zoning rules — and build evidence for policy reform.
Developers & Planners
Look up permit requirements, height limits, and parking rules for specific sites across cities without reading full PDFs.
Journalists & Researchers
Compare zoning regulations across municipalities with cited sources — find stories in the data about housing supply barriers.
Municipal Staff
Benchmark their own bylaws against peer municipalities. Identify areas where regulations may lag best practices.
Citizens
Understand what's allowed in their neighbourhood — whether a secondary suite is possible, what the height rules are.
Policy Makers
Track development permit trends over time and connect zoning rules to actual permit activity on the ground.
